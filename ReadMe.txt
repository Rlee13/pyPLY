Introduction
============

pyPLY is a set of Python classes designed for composite materials calculations such as micromechanics,
laminate stresses and strain and failure indexes based on the Classical Lamination Theory (CLT). 

History
=======

The project started as a part of a master thesis but never got into the latest
version. Some years later, cleaned up, dusted off, here it is...

Requirements
============

*Python ver. >= 2.6 .
*Numpy .
*matplotlib (optional) .
*IPython ver >= 0.13. (optional) .

Instalation
============

There is no package for this version. 
Copy the library's files in the local directory or in the Python's "site-packages" folder.
